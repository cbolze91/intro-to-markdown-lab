## 1. What is a Basic HTML Boilerplate

An **HTML boilerplate** is the minimum required code that every HTML file needs in order to work correctly in the browser.

Think of it as the **skeleton** of a webpage — everything else gets added inside of it.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
```


## 2. What Each Part Means

a.  **Doctype**
``` HTML
<!DOCTYPE html>
```
Tells the browser you’re using HTML5.

 b. **The html element**

```HTML
<html lang="en"></html>
```
Wraps the whole document and sets the language.

c. **The head section**
```HTML
<head> ... </head>
```
Holds “behind-the-scenes” info like title, character encoding, CSS links, etc.

d. **Character encoding**
```HTML
<meta charset="UTF-8">
```
Makes text render correctly (symbols, emojis, special characters).

e. **Title**
```HTML
<title>My Web Page</title> 
```
Shows in the browser tab.

f. **The body section**
```HTML
<body> ... </body>
```
Holds everything visible on the page (text, images, buttons, etc.).
